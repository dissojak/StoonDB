# Security Policy

## Supported Versions

Security fixes are provided for the latest version on `main`.

## Reporting a Vulnerability

If you discover a security issue, please report it privately to project
maintainers before opening a public issue.

Include:

- Description of the issue
- Reproduction steps
- Potential impact
- Suggested mitigation if known

## Security Considerations

- This app runs local service commands.
- It should be used only on trusted local development machines.
- Avoid committing secrets or credentials into repository files.
